# House Price Prediction using Machine Learning

This project predicts house prices using machine learning techniques.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Models Implemented
- Linear Regression (Baseline Model)
- Random Forest Regressor (Improved Model)

## Workflow
1. Data loading and exploration
2. Data visualization and correlation analysis
3. Feature and target selection
4. Model training and evaluation
5. Performance comparison

## Results
- Linear Regression R2 Score: ~0.57
- Random Forest R2 Score: ~0.80

## Conclusion
Random Forest significantly improved prediction accuracy compared to Linear Regression.
